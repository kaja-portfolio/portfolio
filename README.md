## Getting Started

1. **Download or Clone** all files to your local machine
2. **Open** `index.html` in your web browser
3. **Customize** the content, colors, and information to match your personal brand

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    --accent-color: #ec4899;
    /* ... more colors ... */
}
```

### Content
- Update personal information in `about.html`
- Add your projects in `projects.html`
- Modify skills and percentages in `skills.html`
- Update contact information in `contact.html`

### Images
Replace the placeholder profile image in `about.html` and project images in `projects.html` with your own.
